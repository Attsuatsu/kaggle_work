# 🚀 Kaggle_work

このリポジトリは、現在制作途中の**Kaggle分析プロジェクト集**です。  
現段階では、下記の2つの分析ワークを公開しています。  
いずれも学習・実践を通して、**データ分析の理論理解と応用力の向上**を目的としています。

---

## 📂 フォルダ構成

### 1. **時系列解析_店舗商品需要予測**
- **使用データ**: Kaggle主催「Store Item Demand Forecasting Challenge」  
- **目的**: 大会は終了していましたが、**時系列解析の理解を深めるため**大学の課題として取り組みました。  
- **成果**: 本課題では**満点を獲得（56人中12人のみ）**。  
- **内容**:  
  - **R言語**を用いて解析を実施（統計的手法を中心に採用）。  
  - データ前処理、季節性やトレンドの分析、ARIMAモデルなどを用いた予測。  
  - 分析結果と考察をPDFにまとめています。  
- **学んだこと**:  
  Rによる統計的アプローチを通して、**時系列データの構造理解とモデリングの奥深さ**を実感しました。

---

### 2. **決定木_交通事故リスク予測**
- **使用データ**: Kaggleコンペ「Predicting Road Accident Risk」  
- **参加状況**: 開催期間中に参加し、**1671 / 4082位**を獲得。  
- **内容**:  
  - 決定木ベースのモデル（LightGBM等）を用いた事故リスク予測。  
  - 分析過程をJupyter Notebookで整理。  
- **学んだこと**:  
  特徴量重要度の分析やモデル解釈を通じて、**精度と説明性の両立**の大切さを学びました。
